# OpenProxy UI v0.1

## Features 

1. 로컬 룰을 생성   
2. 리모드 룰을 표시 
3. 리모트 룰을 로컬 룰로 복사 
4. 로컬 룰은  구조 변경 
5. 룰 설정 창  (file, folder, status 포함)
6. 창을 닫으면 기본 데이타 모두 저장
